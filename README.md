# Fortune Coach

AI 私人命理師 + 每日行動教練。

算命、星座、塔羅與行動建議結合的手機版 Web App 原型。

## Project Goal

Fortune Coach 不是傳統算命 App。

它的目標是把星座、塔羅、紫微、風水等素材，轉換成使用者每天看得懂、做得到、願意回來看的生活建議。

核心感覺：

> 像一位懂命理的 AI 顧問，也像每天提醒你往前走一步的行動教練。

## First Version Direction

第一版先做手機版 Web App。

先驗證：

- 使用者會不會每天打開
- AI 命理師的語氣是否有陪伴感
- 每日行動建議是否真的讓人想照做
- 使用者是否願意留下「今天準不準」回饋

## First Version Inputs

第一版只問最少資料：

- 生日
- 星座
- 最近最關注的主題

關注主題先用選項：

- 工作
- 感情
- 財運
- 健康
- 人際
- 自我成長

可加一題自由輸入：

> 最近有什麼特別想問命運的事？

## First Version Screens

1. 今日首頁
2. 每日塔羅
3. AI 私人命理師聊天
4. 每日行動教練
5. 晚間運勢驗證

## Product Principle

第一版不追求玄學系統完整。

第一版只追求一件事：

> 使用者每天打開後，覺得它懂我，而且今天真的可以照它說的做一點點。
