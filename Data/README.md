# Data

放 Fortune Coach 的文案資料與規則資料。

第一版最重要的是每日行動教練 wording database。

v1.0 已加入塔羅牌資料庫：

- `tarot_cards_v1_0.json`
- `tarot_database_schema.md`

建議資料來源：

- `daily_action_wording.md`
- `wording_database_schema.md`
- `tarot_cards_v1_0.json`
- 日後可轉為 `fortune_wording.xlsx`

原則：

文案不要寫死在 App 裡。

每日行動建議要可以新增、修改、停用、分類。
