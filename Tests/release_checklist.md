# Release Checklist

每次輸出 prototype 或版本前確認：

## Mobile

- iPhone Safari 可閱讀
- Android Chrome 可閱讀
- 首頁第一屏沒有文字重疊
- 按鈕夠大，手指容易點
- 深色背景下文字清楚
- 主視覺不壓過主要資訊

## Core Flow

- 第一次使用可以填生日
- 第一次使用可以選星座
- 第一次使用可以選關注主題
- 今日首頁正常顯示
- 今日行動建議正常顯示
- AI 命理師入口明顯
- 每日塔羅入口明顯
- 晚間回饋入口容易找到

## Wording

- 沒有恐嚇語氣
- 沒有保證式預測
- 沒有過度命定論
- 建議是可執行的
- 文字像真人命理師，不像罐頭星座文

## Scope

- 沒有偷加社群
- 沒有偷加 AR
- 沒有偷加複雜命盤
- 沒有把文案大量寫死在程式裡
