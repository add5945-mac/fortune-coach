# Homepage Concept

## Goal

首頁第一眼要讓使用者覺得：

> 這不是普通星座 App，這是一位今天會陪我做決定的 AI 命理師。

## First Screen

手機第一屏包含：

- 背景主視覺：羅盤 + 塔羅 + 星空
- App 名稱：Fortune Coach
- 今日關鍵字
- 今日一句話
- 今日運勢分數
- 工作 / 感情 / 財運 / 狀態
- 主要按鈕：問 AI 命理師
- 次要按鈕：抽今日塔羅

## Example Copy

```text
Fortune Coach

今日關鍵字
順勢而為

今天不是硬衝的一天。
先觀察，再出手，反而更容易得到好結果。

今日運勢 78

工作 ★★★☆☆
感情 ★★★★☆
財運 ★★★★★
狀態 ★★★☆☆
```

## Second Section

每日行動教練。

```text
今日行動建議

1. 重要訊息晚一點再回
2. 今天適合整理財務或檢查帳單
3. 下午適合主動聯絡一位貴人
```

## Third Section

AI 私人命理師聊天入口。

```text
你今天可以問我：

今天適合談加薪嗎？
今天適合告白嗎？
我最近工作很煩，今天要注意什麼？
這個人今天適合聯絡嗎？
```
